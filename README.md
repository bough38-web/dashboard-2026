# 2026 Management Hub - 관리고객 재계약 관리 시스템

2026년 관리고객 재계약 활동 및 리텐션을 관리하기 위한 프리미엄 대시보드 시스템입니다.

## 📄 주요 문서 (Project Documentation)
*   **[프로젝트 기술 평가 및 보고서 (Project Report)](https://bough38-web.github.io/dashboard-2026/PROJECT_REPORT.html)**
*   **[시스템 사용 매뉴얼 (User Manual)](https://bough38-web.github.io/dashboard-2026/USER_MANUAL.html)**

## 🚀 주요 기능 (Key Features)
- **실시간 KPI 대시보드**: 만기 신규, 재계약, 약정 만료 실적 현황 시각화
- **GIS 지도 관제**: 구글 맵 기반 고객 위치 및 활동 데이터 매핑
- **프리미엄 UI/UX**: 4종 테마(Default, Dark, Emerald, Sunset) 및 반응형 레이아웃 지원
- **데이터 자동화**: 엑셀 데이터를 웹 대시보드 데이터로 변환하는 파이프라인

## 💻 실행 방법 (How to Run)
1.  이 저장소를 클론합니다.
2.  로컬 환경에서 웹 서버를 실행합니다 (예: `python3 -m http.server 8000`).
3.  브라우저에서 `login.html`로 접속합니다 (관리자 코드: `0303`).
